# Cloud-Based File Backup & Recovery App

## Overview
This project allows users to upload files and store them securely in the cloud with version control and restore options.

## Core Features
- File Upload
- Cloud Storage (Firebase)
- Version History
- File Recovery / Download

## Tools & Technologies
- HTML, CSS, JavaScript
- Firebase Storage
- Firebase Authentication
- Firebase Firestore (optional)
